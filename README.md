# qinajia-mern-project

## 技術線

前端：HTML、CSS、JavaScript、React、Next.js、Tailwind CSS
後端：Node.js、MongoDB Atlas
部署：前端 Vercel, 後端 Heroku

## 功能

- 投票功能
- 占卜 (依據主題隨機給一段話)
- 幹話 (依據主題隨機給一段話，可添加)
- ws聊天室 (目前使用原生websocket，本地可用，線上要轉socketio等)
- 登入、修改會員資料
- 關於網站
- 聯絡我們
